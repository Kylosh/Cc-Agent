=============================================================
REACT FRONTEND SOURCE CODE - COMPLETE FILE STRUCTURE
=============================================================
Project: Fruit Delivery Frontend
Framework: React 18.2.0 with React Router v6
State Management: React Hooks (useState, useContext)
Styling: CSS Modules + Inline Styles

=============================================================
CORE FILES CREATED/UPDATED
=============================================================

1. src/index.js
   - React entry point
   - Renders App component into root DOM element
   - Uses React.StrictMode for development warnings

2. src/App.js
   - Main application component
   - Implements React Router with 4 routes:
     * "/" -> Home page
     * "/products" -> Product listing page
     * "/product/:id" -> Product detail page
     * "/checkout" -> Checkout page
   - Global cart state management using useState + useEffect
   - localStorage persistence for cart data
   - Includes Header, ShoppingCart sidebar, and Footer
   - Methods:
     * handleAddToCart() - adds/updates products in cart
     * handleRemoveFromCart() - removes items from cart
     * handleUpdateQuantity() - updates item quantities
   - Cart count calculation for badge display

3. src/components/Header.js
   - Navigation header component
   - Features:
     * Logo with branding
     * Navigation links (Home, Products, Features, About)
     * Shopping cart button with item count badge
     * Mobile hamburger menu toggle
   - Props: cartCount, onCartClick

4. src/components/ProductCard.js
   - Reusable product card component
   - Displays:
     * Product image/emoji
     * Product name and description
     * Current price and original price (with discount)
     * "Add to Cart" button overlay on hover
   - Links to product detail page
   - Props: product, onAddToCart

5. src/components/ShoppingCart.js
   - Sidebar shopping cart component
   - Features:
     * Show/hide toggle with overlay backdrop
     * List of cart items with quantities
     * Quantity increase/decrease buttons
     * Remove item functionality
     * Empty cart state with "Continue Shopping" link
     * Order summary (subtotal, shipping, total)
     * Checkout button
   - Props: items, isOpen, onClose, onUpdateQuantity, onRemove

6. src/pages/Home.js
   - Landing/home page
   - Features:
     * Hero section (handled by Hero component)
     * Featured products section (4 products)
     * Features showcase section (4 feature cards)
     * Call-to-action section
   - Props: onAddToCart

7. src/pages/Products.js
   - Product listing page
   - Features:
     * Product grid display (8 products with categories)
     * Sorting functionality:
       - Popularity (default)
       - Price (low to high)
       - Price (high to low)
       - Name (alphabetical)
     * Product count display
     * Responsive grid layout
   - Props: onAddToCart
   - State: sortBy (current sort method)

8. src/pages/ProductDetail.js
   - Individual product detail page
   - Features:
     * Product image with emoji display
     * Product name, rating, and reviews
     * Price display with discount calculation
     * Detailed product information
     * Quantity selector (1-99 range)
     * Add to cart button
     * Shipping information display
   - Props: onAddToCart
   - URL param: id (product ID)
   - State: quantity

9. src/pages/Checkout.js
   - Checkout/order completion page
   - Features:
     * Order summary with line items
     * Shipping cost calculation
     * Tax calculation (10% of subtotal)
     * Delivery form with fields:
       - Name, Email, Phone
       - Address, City, Postal Code
     * Form validation (required fields)
     * Processing state during submission
     * Success screen after order placement
     * Empty cart error handling
   - Props: cartItems
   - State: formData, isProcessing, orderPlaced

10. src/index.css
    - Global styles and CSS variables
    - Root variables defined
    - Base element styles
    - Utility classes
    - Responsive design helpers

=============================================================
KEY FEATURES & PATTERNS
=============================================================

1. React Hooks Usage:
   - useState: for component state management
   - useEffect: for side effects and localStorage sync
   - useParams: for URL parameters
   - useNavigate: for programmatic navigation

2. State Management:
   - Cart state in App.js (parent component)
   - Props drilling for callbacks
   - localStorage for cart persistence

3. Routing:
   - React Router v6
   - Dynamic routes with URL parameters
   - Link components for navigation

4. Form Handling:
   - Controlled components in Checkout.js
   - Form validation with required fields
   - onChange handlers with state updates

5. Responsive Design:
   - Mobile-first approach
   - CSS media queries
   - Flexible grid layouts

=============================================================
DATA FLOW ARCHITECTURE
=============================================================

App.js (Cart State)
  |
  |-> Header.js (receives: cartCount, onCartClick)
  |
  |-> ShoppingCart.js (receives: items, isOpen, callbacks)
  |
  +-> Routes:
      |-> Home.js
      |   +-> ProductCard.js (receives: product, onAddToCart)
      |
      |-> Products.js
      |   +-> ProductCard.js (multiple, with filtering/sorting)
      |
      |-> ProductDetail.js (receives: onAddToCart)
      |
      +-> Checkout.js (receives: cartItems)
