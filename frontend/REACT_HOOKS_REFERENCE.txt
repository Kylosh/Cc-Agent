1. CART STATE MANAGEMENT (App.js)
----------------------------------
const [cartItems, setCartItems] = useState([]);
const [isCartOpen, setIsCartOpen] = useState(false);

useEffect(() => {
  const savedCart = localStorage.getItem('cart');
  if (savedCart) {
    try {
      setCartItems(JSON.parse(savedCart));
    } catch (err) {
      console.error('Failed to load cart:', err);
    }
  }
}, []);

useEffect(() => {
  localStorage.setItem('cart', JSON.stringify(cartItems));
}, [cartItems]);

2. ADD TO CART LOGIC (App.js)
------------------------------
handleAddToCart(product) {
  - Check if product already exists in cart
  - If exists: increment quantity
  - If new: add product with quantity = 1
  - Auto-open cart sidebar
}

3. REMOVE FROM CART (App.js)
----------------------------
handleRemoveFromCart(productId) {
  setCartItems(prev => prev.filter(item => item.id !== productId));
}

4. UPDATE QUANTITY (App.js)
---------------------------
handleUpdateQuantity(productId, newQuantity) {
  - If newQuantity <= 0: remove from cart
  - Otherwise: update item quantity
}

ROUTING STRUCTURE (App.js with React Router v6)
===============================================

Routes:
1. / -> Home component
2. /products -> Products component
3. /product/:id -> ProductDetail component
4. /checkout -> Checkout component

Navigation:
- Link components throughout for navigation
- useNavigate hook in Checkout for navigation after order
- useParams hook in ProductDetail for product ID

FORM HANDLING (Checkout.js)
==========================

Form Fields:
- name (text, required)
- email (email, required)
- phone (tel, required)
- address (text, required)
- city (text, required)
- zipcode (text, required)

Validation:
- HTML5 required attribute
- Form submission prevents default
- Processing state during submission

Success Flow:
1. User fills form and clicks submit
2. isProcessing = true (button disabled, loading state)
3. Simulate API call with setTimeout (1500ms)
4. orderPlaced = true (show success screen)
5. Display order number (Date.now())
6. Show shipping info
7. "Return to Home" button to navigate back

PERFORMANCE CONSIDERATIONS
==========================

1. useEffect dependencies:
   - Cart persistence only runs when cartItems change
   - Prevents infinite loops

2. Props drilling:
   - onAddToCart passed down to child components
   - More prop drilling could be replaced with useContext

3. Memoization:
   - ProductCard could use React.memo for optimization
   - Not critical for this app size
